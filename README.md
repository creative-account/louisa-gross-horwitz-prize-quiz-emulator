# ルイザ・グロス・ホロウィッツ賞クイズエミュレーター

## このアプリについて
ルイザ・グロス・ホロウィッツ賞日本人受賞者を覚えるならこのアプリ！  
QuizKnockさんの動画、[東大生100点は取り飽きたので人生最高得点を取る旅に出ます](https://www.youtube.com/watch?v=AISo8I4xYL4)
において、ふくらPが1不可説不可説転点を取るために作成したアプリの再現です。  
2026年2月現在、元のアプリは削除されているようです。

なお、元のアプリではタイトル画面のフォントにフォントワークスの「くろかね」を使用しているようですが、フォントワークスLETSではWebフォントとしての使用ができないので、再現したい方はくろかねをインストールし、各自で変更してください。
